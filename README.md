\# 🔍 AI Log Analyzer



A Django-based web application that analyzes system logs using rule-based detection and machine learning (Isolation Forest).



\## 🚀 Features

\- Upload log files

\- Detect brute-force attacks

\- Identify suspicious IP activity

\- AI-based anomaly detection

\- Clean dashboard UI



\## 🛠 Tech Stack

\- Python

\- Django

\- Scikit-learn

\- Bootstrap



\## ▶️ Run Locally

```bash

pip install -r requirements.txt

python manage.py runserver

```
